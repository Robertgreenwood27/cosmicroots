# Product Images

Add your product images here following this structure:

```
public/images/
├── glassware/
│   ├── galaxy-pipe.jpg
│   ├── sunset-mesa-bubbler.jpg
│   └── ...
├── incense/
│   ├── desert-sage.jpg
│   ├── palo-santo.jpg
│   └── ...
├── tapestries/
│   ├── sacred-geometry.jpg
│   ├── psychedelic-forest.jpg
│   └── ...
└── stickers/
    ├── third-eye.jpg
    ├── stay-cosmic.jpg
    └── ...
```

Image paths in the JSON data files reference these locations.

For best results, use:
- **Format**: JPG or PNG
- **Size**: 800x800px minimum
- **Quality**: High resolution, well-lit
- **Style**: Consistent backgrounds (white or neutral)

You can use placeholder images for now, or add real product photos as you build out your catalog.
